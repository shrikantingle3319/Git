# Simple Interest Calculator (Python)

## 📌 Project Description

This project is a **Simple Interest Calculator built using Python**.
It allows the user to calculate the **simple interest and total amount** based on the given **principal amount, rate of interest, and time period**.

This program is useful for beginners who want to understand **basic Python programming, user input, and mathematical calculations**.

---

## 🧮 Formula Used

The formula used to calculate Simple Interest is:

Simple Interest = (Principal × Rate × Time) / 100

Where:

* **Principal (P)** = Initial amount of money
* **Rate (R)** = Rate of interest per year
* **Time (T)** = Time period in years

---

## ⚙️ Features

* Takes **user input** for principal, rate, and time
* Calculates **simple interest**
* Calculates **total amount payable**
* Easy to understand **beginner-friendly Python code**
